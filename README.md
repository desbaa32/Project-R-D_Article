# UCRNN - Classification de Sentiments sur Réseaux Sociaux

Ce projet implémente et compare différents modèles de classification de sentiments pour les textes de réseaux sociaux, en particulier dans des conditions de données limitées.

## Modèles Implémentés

1. **TextCNN** : Modèle CNN classique pour la classification de texte
2. **UCRNN** : Modèle hybride combinant Bi-LSTM pour le texte et CNN pour les attributs utilisateur
3. **UCRNN+** : Version améliorée avec mécanismes d'attention
